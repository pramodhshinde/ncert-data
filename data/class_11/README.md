# Class 11 — Stream Folders and Subject Mapping

This directory corresponds to Class 11. All subjects/matter are organized by "stream" subfolders (science, commerce, arts, common) per app contract.

**See below for mapping and directory explanation.**

---

## Folder Layout (per NCERT App Data Contract)

- `science`    — Physics, Chemistry, Biology, Mathematics, Computer Science, Biotechnology, Physical Geography, Health & PE, Psychology
- `commerce`   — Business Studies, Financial Accounting 1/2, Indian Economic Development, Statistics, Informatics Practice
- `arts`       — History, Political Theory, Understanding Society, Introducing Sociology, Indian Constitution at Work, Art
- `common`     — English, Hindi, Woven Words, Hornbill, English Snapshots

## Required File Types (every stream folder):

- `books.json`          — Book list for subjects in this stream
- `chapters.json`       — Chapters per book metadata
- `solutions.json`      — Per-chapter solutions metadata (can be empty)
- `notes.json`          — Per-chapter notes metadata (can be empty)
- `mcqs.json`           — All MCQs (one per subject or empty)
- `mcqs/`               — Directory: per-chapter MCQ JSONs
- `papers.json`         — Past paper list (for science, commerce)
- `paper_subjects.json` — Paper subject list (for science, commerce)
